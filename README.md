# Personal Website

A self-contained personal website starter built with plain HTML, CSS, and JavaScript.

## Files

- `index.html`: page structure and content
- `styles.css`: layout, responsive styling, and visual design
- `script.js`: mobile navigation, active section highlighting, and reveal animations

## Customize

1. Replace `Your Name`, `Your City, Country`, and the placeholder bio text in `index.html`
2. Update the project cards, experience timeline, and contact links
3. Open `index.html` in a browser to preview locally

## Deploy

You can deploy these files directly to GitHub Pages, Netlify, Vercel static hosting, or any basic web host.
